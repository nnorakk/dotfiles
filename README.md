# dotfiles
Meus (dotfiles|Arquivos de configuração)...um projeto inacabado

## Ambiente(s) Utilizado(s)
### Sistema Operacional
* Fedora Linux ( maior conteúdo )
* Macos ( Pouca utilização )
* Arch Linux ( futuro)

### Principais softwares
* bspwm ( gerenciador de janelas )
* zsh ( shell )
* vim ( editor de arquivos )

## Instalação
* Clonar repositorio

```bash
git clone https://github.com/nnorakk/dotfiles ~/.dotfiles
```
* Executar script de inicialização

```bash
cd ~/.dotfiles/init && source bootstrap.sh
```
